# Brandstore - E-commerce Website

A complete e-commerce website replica of Brandstore-02, built with HTML, CSS, and JavaScript.

## Features

- **Home Page** - Hero banner, client logos, category banners, featured products, and special edition section
- **Store Page** - Full product listing with filters and sorting
- **Category Pages** - Women, Men, and Accessories category pages
- **Product Detail Page** - Individual product pages with image gallery and options
- **About Page** - Company information
- **Contact Page** - Contact form
- **Shopping Cart** - Functional cart with localStorage persistence
- **Responsive Design** - Mobile-friendly layout
```

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the pages using the navigation menu
3. Add products to cart
4. View cart by clicking the cart icon in the header

## Pages

- **Home** (`index.html`) - Main landing page
- **Store** (`store.html`) - All products with filters
- **Women** (`category-women.html`) - Women's products
- **Men** (`category-men.html`) - Men's products
- **Accessories** (`category-accessories.html`) - Accessories
- **Product Detail** (`product-detail.html?id=1`) - Individual product view
- **About** (`about.html`) - About us page
- **Contact** (`contact-us.html`) - Contact form

## Features

### Shopping Cart
- Add products to cart
- View cart in sidebar
- Persistent cart (saved in localStorage)
- Remove items from cart
- View total price

### Product Filtering
- Filter by price range
- Sort by price, popularity, rating, latest
- Category filtering

### Responsive Design
- Mobile-friendly layout
- Tablet optimization
- Desktop experience

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- All images are stored in the `images/` folder
- Cart data is saved in browser localStorage
- No backend required - fully static website
